# LangGraph Tutorials

Welcome to the **LangGraph Tutorials** repository! This project contains a collection of Jupyter Notebooks demonstrating how to build advanced AI workflows, prompt chains, and agents using **LangChain** and **LangGraph**.

## 🛠️ Tech Stack
- **Python**
- **LangChain & LangGraph**
- **Groq API (Llama 3)** 

## ⚙️ Setup & Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Akshat-124/langgraph.git
   ```
2. Create a virtual environment and activate it:
   ```bash
   python -m venv myenv
   myenv\Scripts\activate
   ```
3. Install the required dependencies:
   ```bash
   pip install langchain-groq langgraph python-dotenv jupyter
   ```
4. Create a `.env` file in the root directory and add your Groq API Key:
   ```env
   GROQ_API_KEY=your_api_key_here
   ```

## 🧠 Usage
Run Jupyter Notebook or Jupyter Lab to explore the workflows:
```bash
jupyter notebook
```
